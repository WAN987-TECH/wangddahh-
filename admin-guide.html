团长操作手册

本手册适合已经完成首次部署的团长，介绍每次团购的完整操作流程和常用功能。

一、登录后台

浏览器打开 你的网址#admin，输入管理密码进入后台。

后台左侧有五个功能标签：商品管理、订单汇总、收款清单、下单二维码、轮次设置。

二、创建与管理轮次

1

左侧菜单 → 轮次设置 → 点击「创建新轮次」

  • 填写轮次名称（如「5 月第一周团购」）
  • 设置下单截止时间（选填，到期后顾客端自动显示「已截止」并关闭下单）
  • 点击「创建」，轮次进入「策划中」状态
轮次有四种状态：
· 策划中 — 刚创建，不可下单，顾客端看不到
· 进行中 — 已开团，顾客可浏览商品和下单
· 已截止 — 到达截止时间自动变更，不可再下单
· 已停止 — 团长手动停止，等同已截止
允许多个轮次同时「进行中」，不同轮次的商品和订单完全隔离，适合水果团、日用品团同时开。

添加商品完成后，点击该轮次的「开团」按钮使其变为「进行中」，顾客端才可见。

进行中可点「停止」提前结束；已停可点「启用」重新开团。只有已停止的轮次才能「删除」（会同时删除该轮次所有订单，不可恢复）。

三、添加商品

2

左侧菜单 → 商品管理 → 选择轮次 → 点击「添加商品到「轮次名」」

  • 商品名称（必填）— 如「烟台红富士苹果」
  • 商品图片（选填）— 支持 JPG/PNG/GIF/WebP，最大 5MB，可直接拍照上传
  • 规格与价格（至少一个）— 如「500g / 15 元」「1kg / 28 元」，可添加多个规格
  • 整装拆分计算器 — 适用于山姆、开市客等大包装商品。输入整装数量（如 30 个）、总价（如 19.90 元）、每份大小（如 5 个），一键自动算好份数和单价
  • 商品标签（选填）— 如「蛋糕 冷藏」，用空格分隔，顾客端可按标签筛选
  • 库存数量（选填）— 不填则不限，卖完自动显示「已售罄」并停止加购
  • 点击「添加商品」，商品默认上架
拆分商品命名格式为「5/30个/份」，表示从 30 个一盒中拆成每份 5 个出售。顾客端按「份」显示和购买，库存按「份」扣减。

四、开团与生成二维码

3

商品添加完毕后，到 「轮次设置」 点击该轮次的「开团」按钮。

然后进入 「下单二维码」 标签页：

  • 二维码自动生成，指向顾客下单页面
  • 长按二维码图片 → 保存到手机相册
  • 发到业主微信群，配上文字说明（如「本周团购开始啦，大家扫码下单」）
  • 也可点击「复制下单链接」直接发链接到群里

五、查看与处理订单

4

左侧菜单 → 订单汇总

  • 顶部下拉选择不同轮次，切换后订单列表自动刷新
  • 顶部统计卡片显示当前轮次的订单数和总金额
  • 按人分组显示订单明细（商品、规格、数量、金额、备注）
  • 底部「采购清单」按商品汇总采购数量，可一键复制到美团优选等平台统一采购
  • 点击「导出 Excel」下载含「订单明细」和「商品汇总」两个 sheet 的表格

砍单功能

对有问题的订单（如邻居下错、临时不要），点击订单下方的「砍单」按钮：

  • 弹出确认弹窗，可填写砍单原因（选填，如「缺货」「规格不符」）
  • 确认后订单取消，库存自动恢复
  • 顾客查单时会看到红色「已砍单」标签和砍单原因
  • 区别于顾客自行取消（显示灰色「已取消」）

六、收款

5

左侧菜单 → 收款清单

  • 设置本轮费用（运费、包装费、其他费用），系统自动按人数均摊到分
  • 底部「收款清单」列出每人应付金额：商品金额 + 均摊费用
  • 点击「一键复制收款清单」
  • 打开微信 → 业主群 → 群工具 → 群收款，参考清单逐一发起

七、常见操作

编辑商品

商品管理 → 点击商品右侧「编辑」→ 修改后点「保存修改」。

如果该商品已有顾客订单，规格名称会被锁定不可修改（防止数据不一致），只能修改价格、库存、标签等。

上下架商品

商品管理 → 点击「下架」隐藏商品，顾客端不可见。再点击「上架」恢复显示。

删除商品

商品管理 → 点击「删除」→ 确认。该商品会从所有订单中标记为「已删除」，金额归零，不影响其他商品。

导入导出商品

商品管理 → 点击「导出商品」下载当前轮次商品到 Excel。点击「导入商品」选择之前导出的文件批量恢复商品。适合轮次间复用商品配置。

整装拆分计算器

在添加/编辑商品弹窗中，展开「整装拆分自动计算」：

  • 填入整装数量（如一盒 30 个）、单位（如「个」）、总价（如 19.90 元）、每份大小（如 5 个）
  • 点击「一键添加」，自动生成规格如「5/30个/份」，单价 3.32 元/份
  • 提示盈亏差额和余数信息，方便核对

修改站点标题

轮次设置 → 顾客端标题输入框 → 输入新标题 → 保存。顾客端顶部会实时更新。

修改管理密码

轮次设置 → 管理密码区域 → 输入新密码（≥4 位）→「修改密码」。

忘记密码

在 Supabase → SQL Editor 执行：

UPDATE settings SET value = '' WHERE key = 'admin_password';

然后重新打开 #admin 设置新密码。

八、小贴士

  • 售罄商品会自动沉到列表底部，显示「已售罄」标记，顾客无法加购
  • 库存会每 10 秒自动刷新，多设备同时下单不会超卖
  • 顾客下单后在页面顶部「查/改订单」可自行查询和取消（截止前）
  • 顾客查单时会看到订单所属轮次和状态，被砍单会显示红色砍单原因
  • 轮次切换时顾客购物车会自动清空，防止跨轮次商品混淆
  • 允许多个轮次同时「进行中」，适合不同品类分开团购
  • 商品可导出为 Excel 再导入到新轮次，无需重复填写
  • 历史轮次可在订单汇总下拉中查看，数据不会丢失
